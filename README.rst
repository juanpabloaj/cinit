=====
cinit
=====

Create a c/c++ directory structure.

Install
=======

::

  pip install cinit

Usage
=====

To create a src and include directory with a Makefile
::

    $ cinit
    Creating directory: src
    Creating directory: include
    Generating Makefile
    $ tree
    .
    ├── Makefile
    ├── include
    └── src
